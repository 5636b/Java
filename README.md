# Java

## JavaPoo 1

### EXERCÍCIO DE FIXAÇÃO 1

1.​ Faça um programa em Java que escreva a mensagem "Alô mundo".

2.​ Faça um programa em Java que leia do usuário um número e escreva a mensagem: O número informado foi <número>.

3.​ Faça um programa em Java que leia do usuário dois números e calcule a soma.

4.​ Faça um programa em Java que leia do usuário as 4 notas de um aluno e calcule a média.

5.​ Faça um programa em Java que leia do usuário o número de cavalos de um haras e calcule quantas ferraduras devem ser compradas.

6.​ Faça um programa em Java que leia do usuário um valor em metros e converta para centímetros.

7.​ Faça um programa em Java calcule a área de um quadrado, em seguida escreva o dobro desta área.

8.​ Faça um programa em Java que leia do usuário quanto ele ganha por hora e o número de horas trabalhadas no mês. Calcule e mostre o total do seu salário no referido mês.

9.​ Tendo como dados de entrada a altura de uma pessoa, construa um programa em Java que calcule seu peso ideal, usando a seguinte fórmula: (72.7*altura) – 58.

10.​Faça um programa em Java que leia do usuário a temperatura em graus Fahrenheit e converta a temperatura para graus Celsius. Utilize a seguinte fórmula: C = (5 * (F-32) / 9).
